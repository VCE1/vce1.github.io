alist


简介: 一个支持多种存储的文件列表程序，使用 Gin 和 Solidjs。

项目链接：https://github.com/alist-org/alist

alist 几乎支持挂载市面上所有主流的云盘，通过挂载网盘可以实现一些很酷的操作。比如访问小雅超集，实现随时随地的看剧，非常方便。

ddns-go


ddns-go 可以解析你的动态公网v4/v6到你的域名，访问的时候只需要域名+端口即可。

如果你家里有nas，并且你想要实现外网访问，那么这么工具是必需的。

frp


简介：一款高性能的反向代理工具，支持 TCP、UDP、HTTP、HTTPS 等协议。

项目链接：https://github.com/fatedier/frp

frp 是一个功能强大的反向代理工具，可以帮助你轻松地穿透内网，实现从外部访问内网服务的需求。无论是远程控制、内网穿透还是其他场景，frp 都是一个可靠的选择。

如果你想外网访问NAS，而且恰好有一台vps，且访问速度不错，可以考虑使用frp内网穿透。

jellyfin


简介：一款用于管理和播放多媒体文件的媒体服务器软件，支持电影、音乐、电视节目等。

项目链接：https://github.com/jellyfin/jellyfin

jellyfin 是一个出色的媒体服务器，让你能够轻松地在各种设备上流式传输和管理你的媒体文件。不管是家庭影院还是随时随地的娱乐，jellyfin 都提供了出色的用户体验。

在window系统上安装potplayer并使用：Potplayer插件
在安卓系统上请使用：Jellyfin安卓应用
在IOS上请使用: Infuse

joplin


简介：一款开源的笔记应用，支持 Markdown 格式，具备强大的同步和组织功能。

项目链接：https://github.com/laurent22/joplin

joplin 是一款强大的笔记应用，不仅支持 Markdown 格式，还提供了强大的同步和组织功能。无论是个人笔记还是团队协作，joplin 都是一个非常实用的工具。

komga


简介：一款漫画和电子书管理器，支持多种文件格式。

项目链接：https://github.com/gotson/komga

komga 是一款功能丰富的漫画和电子书管理器，支持多种文件格式。它提供了直观的界面和强大的组织功能，使你能够轻松地管理和阅读你的漫画和电子书收藏。

nginx-proxy-manager


简介：一个简单而强大的 Nginx 反向代理管理工具，使用 Docker 部署方便。

项目链接：https://github.com/jc21/nginx-proxy-manager

nginx-proxy-manager 是一个使 Nginx 反向代理配置变得简单的管理工具。通过使用 Docker 部署，你可以轻松地管理多个域名和服务，并自动处理 SSL 证书。无论是个人项目还是小型企业，都适用于快速部署和管理反向代理。

pingvin share

简介：一个简单的文件共享服务，支持通过浏览器上传和下载文件。

项目链接：https://github.com/stonith404/pingvin-share

pingvin share 提供了简单而高效的文件共享服务，让你可以通过浏览器轻松地上传和下载文件。无需复杂的配置，即可实现便捷的文件分享。

qbitcorrent

简介：一款轻量级的 BitTorrent 客户端，支持多平台。

项目链接：https://github.com/qbittorrent/qBittorrent

qbitcorrent 是一款功能强大且易于使用的 BitTorrent 客户端。它具有跨平台支持、低系统资源消耗等优点，是一个理想的选择，无论是下载大型文件还是管理种子文件。

玩bt的应该都知道的，适合安装在NAS上，配合RSS订阅可以第一时间下载最新的番剧。

rss-to-telegram

简介：将 RSS 订阅内容发送到 Telegram 的工具，方便获取更新。

项目链接：https://github.com/username/rss-to-telegram

rss-to-telegram 是一个方便的工具，可以将你的 RSS 订阅内容直接发送到 Telegram，让你可以随时随地获取最新的信息。适用于那些希望通过 Telegram 轻松跟踪更新的用户。

可以在tg上很方便的查看更新。

wordpress

简介：一款强大的开源博客平台，支持自定义主题和插件。

项目链接：https://wordpress.org/

wordpress 是一款历史悠久且强大的开源博客平台，它支持自定义主题和插件，让你可以轻松创建个性化的博客。无论是个人博客还是专业网站，wordpress 都是一个强大而灵活的选择。

如果要用wordpress建站，推荐使用argon主题*: https://github.com/solstice23/argon-theme

这也是本站点使用的主题，非常简约和美观，可自定义的程度很高。

Telegraph-Image

简介：免费图片托管解决方案，Flickr/imgur替代品。使用Cloudflare Pages和Telegraph。
项目地址：https://github.com/cf-pages/Telegraph-Image

一个免费图床，使用的Telegraph，使用空间不限，需要注意的是上传限制是5M。

因为是白嫖的，本人也在使用。目前也有一些小问题，比如上传的图片在后台可能不显示。需要说明的是，因为是免费，不知道哪天无法使用了，所以还是建议把图片都本地备份一份，预防哪天全部失效了。
